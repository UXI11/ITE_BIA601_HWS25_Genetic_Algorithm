# Intelligent Product Recommendation System (Intelligent RecSys)

An academic project providing smart solutions for product recommendations in e-commerce platforms. It integrates data clustering techniques (K-Means) and Genetic Algorithms (GA) to ensure accurate recommendations that meet user interests.

---

## How to run the project?

To ensure the system works correctly on your device, please follow these simple steps:

### 1. Install required libraries
Open the Terminal in the project folder and install the software requirements via the following command:
```bash
pip install -r requirements.txt
```

### 2. Run the system
After installation, you can run the platform directly using the command:
```bash
streamlit run app.py
```

---

## Important Notes:
* Ensure all Excel files are inside the `data` folder to ensure data is read successfully.
* It is always preferred to start by visiting the Home Page (home icon) when viewing the platform for the first time, to initialize the data in the system memory.
* All interfaces (Store, Analytics, Settings, ...) are in the sidebar for navigation between different project sections.
